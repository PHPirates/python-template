This project uses:

* Python 3
* The unittest test framework
* flake8 for linting
* GitHub Actions for CI

To update dependencies in `requirements.txt`:

* Check venv is activated
* If Python was updated, it may be necessary to remove and recreate the venv
* Run `pip-upgrade`
